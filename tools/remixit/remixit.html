    Remix-It Help

    How to use it

    1. Create the repoless site in Clone-It.
    2. Open Remix-It with the new site name.
    3. Pick the closest industry recipe and adjust the brief.
    4. Choose either the hosted AEMCoder manual path or the direct Catalyst one-shot path.
    5. Review the result in preview, DA, and Bulk app.

    What it does now

    Remix-It supports two consultant workflows: a manual hosted AEMCoder path for today, and a direct one-shot path for Catalyst builds that support the internal window.name launch payload.

    Default safety rule

    Recipes reuse the current block palette by default. Layout, theme, CSS, pacing, and composition are open to redesign; block or template additions remain an escalation path.
